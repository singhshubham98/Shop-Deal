### Development 🔧

- Install Back-end Dependencies.

```sh
npm install
```

- Install Front-end Dependencies.

```sh
npm run client-install
```

- Run Development Server.

```sh
npm run dev
```

- To run only Back-end Server.

```sh
npm run server
```

- To run only Front-end Server.

```sh
npm run client
```
